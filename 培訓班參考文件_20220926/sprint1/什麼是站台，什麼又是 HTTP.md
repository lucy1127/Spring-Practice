# 什麼是站台，什麼又是 HTTP

## 前言

經過了前後端程式的洗禮，可以發現從程式到網頁呈現其實不會太難，但要怎麼讓網頁可以被世界看到就是另外一回事了。這章節，我們會先介紹網頁運作的 HTTP 傳輸協定，再介紹怎麼透過站台讓網站對外公開。

## 課程目標

理解 HTTP 協定的基礎概念，以及架設先前範例撰寫的網站。

作業練習：
使用 Javascript，找一個 Open Data API 進行 HTTP 串接。
